Элементы

grid-column-start, grid-column-end, grid-row-start, grid-row-end

Определяет положение элемента в пределах сетки, ссылаясь на определенные линии сетки. grid-column-start/grid-row-start — это строка, в которой начинается элемент, а grid-column-end/grid-row-end — строка, в которой элемент заканчивается.

Возможные значения:

  • <line> – может быть числом для ссылки на пронумерованную линию сетки или именем для ссылки на именованную линию сетки
  • span <number> – элемент будет охватывать указанное количество дорожек сетки
  • span <name> – элемент будет охватывать до тех пор, пока не достигнет следующей строки с указанным именем
  • auto – указывает на автоматическое размещение, автоматический диапазон или диапазон по умолчанию, равный одному
.item { ⠀⠀⠀⠀grid-column-start: <number> | <name> | span <number> | span <name> | auto; ⠀⠀⠀⠀grid-column-end: <number> | <name> | span <number> | span <name> | auto; ⠀⠀⠀⠀grid-row-start: <number> | <name> | span <number> | span <name> | auto; ⠀⠀⠀⠀grid-row-end: <number> | <name> | span <number> | span <name> | auto; }

Примеры:

.item-a { ⠀⠀⠀⠀grid-column-start: 2; ⠀⠀⠀⠀grid-column-end: five; ⠀⠀⠀⠀grid-row-start: row1-start; ⠀⠀⠀⠀grid-row-end: 3; }
.item-a { ⠀⠀⠀⠀grid-column-start: 1; ⠀⠀⠀⠀grid-column-end: span col4-start; ⠀⠀⠀⠀grid-row-start: 2; ⠀⠀⠀⠀grid-row-end: span 2; }

Если grid-column-end/grid-row-end не объявлен, элемент по умолчанию будет занимать 1 дорожку..

Элементы могут накладываться друг на друга. Вы можете использовать z-index для управления порядком их размещения..

grid-column, grid-row

Сокращение для grid-column-start + grid-column-end и grid-row-start + grid-row-end соответственно.

Возможные значения:

  • <start-line> / <end-line> – каждый принимает все те же значения, что и полные версии свойств, включая диапазон
.item { ⠀⠀⠀⠀grid-column: <start-line> / <end-line> | <start-line> / span <value>; ⠀⠀⠀⠀grid-row: <start-line> / <end-line> | <start-line> / span <value>; }

Пример:

.item-c { ⠀⠀⠀⠀grid-column: 3 / span 2; ⠀⠀⠀⠀grid-row: third-line / 4; }

Если значение конечной строки не объявлено, элемент по умолчанию будет охватывать 1 дорожку.

grid-area

Дает элементу имя, чтобы на него можно было ссылаться в шаблоне, созданном с помощью свойства grid-template-areas. Кроме того, это свойство можно использовать как еще более короткое сокращение для grid-row-start + grid-column-start + grid-row-end + grid-column-end.

Возможные значения:

  • <name> – имя области сетки
  • <row-start> / <column-start> / <row-end> / <column-end> – может быть числом или именованной строкой/столбцом
.item { ⠀⠀⠀⠀grid-area: <name> | <row-start> / <column-start> / <row-end> / <column-end>; }

Примеры:

Присвоение имение элементу:

.item-d { ⠀⠀⠀⠀grid-area: header; }

Сокращение от grid-row-start + grid-column-start + grid-row-end + grid-column-end:

.item-d { ⠀⠀⠀⠀grid-area: 1 / col4-start / last-line / 6; }

justify-self

Выравнивает элемент внутри ячейки сетка вдоль строчной оси (строки) (в отличии от свойства align-self, которое выравнивает вдоль блочной оси (столбец)). Это значение применяется к элементу сетки внутри одной ячейки.

Возможные значения:

  • start – выравнивает элемент сетки так, чтобы он был на одном уровне с начальным краем ячейки
  • end – выравнивает элемент сетки так, чтобы он был на одном уровне с конечным краем ячейки
  • center – выравнивает элемент сетки по центру ячейки
  • stretch – заполняет всю ширину ячейки (по умолчанию)
.item { ⠀⠀⠀⠀justify-self: start | end | center | stretch; }

Примеры:

.item-a { ⠀⠀⠀⠀justify-self: start; }
.item-a { ⠀⠀⠀⠀justify-self: end; }
.item-a { ⠀⠀⠀⠀justify-self: center; }
.item-a { ⠀⠀⠀⠀justify-self: stretch; }

align-self

Выравнивает элемент внутри ячейки сетки по блочной оси (столбец) (в отличии от свойства justify-self, которое выравнивает по строчной оси (строки). Это значение применяется к содержимому внутри одного элемента сетки.

Возможные значения:

  • start – выравнивает элемент сетки так, чтобы он был на одном уровне с начальным краем ячейки
  • end – выравнивает элемент сетки так, чтобы он был на одном уровне с конечным краем ячейки
  • center – выравнивает элемент сетки по центру ячейки
  • stretch – заполняет всю высоту ячейки (по умолчанию)
.item { ⠀⠀⠀⠀align-self: start | end | center | stretch; }

Примеры:

.item-a { ⠀⠀⠀⠀align-self: start; }
.item-a { ⠀⠀⠀⠀align-self: end; }
.item-a { ⠀⠀⠀⠀align-self: center; }
.item-a { ⠀⠀⠀⠀align-self: stretch; }

place-self

place-self устанавливает свойства align-self и justify-self в одном объявлении.

Возможные значения:

  • auto – стандартное значение выравнивания
  • <align-self> / <justify-self> – первое значение устанавливает align-self, второе значение — justify-self. Если второе значение опущено, первое значение присваивается обоим свойствам.

Примеры:

.item-a { ⠀⠀⠀⠀place-self: center; }
.item-a { ⠀⠀⠀⠀place-self: center stretch; }

Все основные браузеры, кроме Edge, поддерживают сокращенное свойство place-self.
