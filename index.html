Терминология

Прежде всего изучите основные термины и обозначения

Новая технология позволяет быстро и удобно создавать гибкую разметку страницы. CSS Grid Layout отлично подходит для того, чтобы разделить страницу на основные области или определить взаимосвязь размера, позиционирования и уровня между частями контента, состоящего из HTML примитивов. При применении CSS Grid ваша разметка не зависит от содержания, так как сетка автоматически подстроится под сетку.
