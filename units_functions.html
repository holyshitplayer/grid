Единицы измерения и функции

Единицы измерения fr

Вероятнее всего, при использовании CSS Grid вы будете часто применять дробные единицы (fr). По сути, они означают «часть оставшегося пространства». Итак, объявление типа:

grid-template-columns: 1fr 3fr;

Значит, приблизительно, 25% 75%. За исключением того, что эти процентные значения не такие гибкие, как дробные единицы. Например, если вы добавили отступы к этим процентным столбцам, то вы нарушили 100% ширину (предполагая блочную модель содержимого). Дробные единицы также намного удобнее в сочетании с другими единицами, как вы можете увидеть:

grid-template-columns: 50px min-content 1fr;

Ключевые размерные слова

При изменении размеров строк и столбцов вы можете использовать все длины, к которым вы привыкли, такие как px, rem, % и т. д., но вы также можете применить ключевые слова:

  • min-content – минимальный размер содержимого. Представьте себе строку текста типа «Привет, мир», минимальное содержание, вероятно, равно ширине слова «Привет».
  • max-content – максимальный размер содержимого. Представьте предложение выше, максимальное содержание — это длина всего предложения.
  • auto – это ключевое слово очень похоже на единицы fr, за исключением того, что они «проигрывают» битву в размерах единицам fr при распределении оставшегося места.
  • fit-content – использует доступное пространство, но не меньше, чем min-content и не больше, чем max-content.
  • fractional units – см. выше

Функции определения размеров

При изменении размеров строк и столбцов вы можете использовать все длины, к которым вы привыкли, такие как px, rem, % и т. д., но у вас также есть функции:

  • Функция minmax() делает именно то, как она называется: она устанавливает минимальное и максимальное значение длины. Это полезно в сочетании с относительными единицами. Например, вы можете захотеть, чтобы столбец мог сжиматься только до сих пор.
    • grid-template-columns: minmax(100px, 1fr) 3fr;
  • Функцияmin().
  • Функция max().

Функция repeat() и ключевые слова

Функция repeat() может сэкономить время при описании свойств:

grid-template-columns: ⠀⠀⠀⠀1fr 1fr 1fr 1fr 1fr 1fr 1fr 1fr; /* будет легче: */ grid-template-columns: ⠀⠀⠀⠀repeat(8, 1fr); /* особенно когда: */ grid-template-columns: ⠀⠀⠀⠀repeat(8, minmax(10px, 1fr));

Но repeat() может стать еще более универсальной функцией в сочетании с ключевыми словами:

  • auto-fill – размещает в строке как можно больше столбцов, даже если они пусты
  • auto-fit – вмещает любые столбцы в пространство. Используйте расширяющиеся столбцы для заполнения пространства, а не пустые столбцы

Это самый известный фрагмент кода во всей CSS Grid и один из величайших CSS-трюков всех времен:

grid-template-columns: ⠀⠀⠀⠀repeat(auto-fit, minmax(250px, 1fr));

Masonry

Экспериментальная функция сетки CSS — masonry. Обратите внимание, что существует множество подходов к CSS masonry, но большинство из них обманчивы и либо имеют серьезные недостатки, либо дают не совсем то, что вы ожидаете.

Теперь у спецификации CSS Grid есть официальный способ, и это используется с флагом в Firefox:

.container { ⠀⠀⠀⠀display: grid; ⠀⠀⠀⠀grid-template-columns: repeat(4, 1fr); ⠀⠀⠀⠀grid-template-rows: masonry; }

Подсетка

Подсетка — чрезвычайно полезная функция сеток, которая позволяет элементам сетки иметь собственную сетку, которая наследует линии сетки от родительской сетки.

.parent-grid { ⠀⠀⠀⠀display: grid; ⠀⠀⠀⠀grid-template-columns: repeat(9, 1fr); } .grid-item { ⠀⠀⠀⠀grid-column: 2 / 7; ⠀⠀⠀⠀display: grid; ⠀⠀⠀⠀grid-template-columns: subgrid; } .child-of-grid-item { ⠀⠀⠀⠀/* получает возможность участвовать в родительской сетке */ ⠀⠀⠀⠀grid-column: 3 / 6; }

На данный момент это поддерживается только в Firefox.

Также полезно знать про display: contents;. Это не то же самое, что подсетка, но иногда она может быть полезным инструментом в похожем стиле.

<div class="grid-parent"> ⠀⠀⠀⠀<div class="grid-item"></div> ⠀⠀⠀⠀<div class="grid-item"></div> ⠀⠀⠀⠀<ul style="display: contents;"> ⠀⠀⠀⠀⠀⠀⠀⠀<!-- Эти элементы сетки участвуют в одной сетке --> ⠀⠀⠀⠀⠀⠀⠀⠀<li class="grid-item"></li> ⠀⠀⠀⠀⠀⠀⠀⠀<li class="grid-item"></li> ⠀⠀⠀⠀</ul> </div>
